# Examples for Periodic Normalization of DDEs

Here we tread two examples of how to use the `PeriodicNormalizationDDEs.jl` package to compute the codimension one periodic normal form coefficients for the fold, period doubling, and Neimark-Sacker bifurcations.

The first example in [NeuralMassModel.md](#NeuralMassModel.md) shows how to compute the periodic normal form coefficients for a fold bifurcation and period doubling bifurcation in a scalar delay differential equation.
